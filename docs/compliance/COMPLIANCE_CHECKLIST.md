# Compliance-Checkliste

**Projekt:** niemehr.de  
**Erstellt:** 2025-11-16  
**Letzte Aktualisierung:** 2025-11-16

---

## Pre-Launch Checkliste (BLOCKIEREND)

### Legal & Compliance

- [ ] Datenschutzerklärung veröffentlicht & verlinkt
- [ ] AGBs veröffentlicht & verlinkt
- [ ] Impressum veröffentlicht
- [ ] Cookie-Banner funktionsfähig
- [ ] VVT erstellt & aktuell
- [ ] Interessenabwägung dokumentiert
- [ ] OpenAI DPA abgeschlossen
- [ ] Aufbewahrungsfristen definiert

### Technical Compliance

- [ ] AI-Disclaimers in allen relevanten UIs
- [ ] Human-in-the-Loop funktioniert
- [ ] User Rights implementiert (Export, Delete)
- [ ] Error Handling ohne Daten-Leaks
- [ ] Logging DSGVO-konform
- [ ] Automatische Löschung konfiguriert

### AI Compliance

- [ ] Prompts enthalten Compliance-Regeln
- [ ] Bias-Testing durchgeführt
- [ ] Edge-Cases getestet
- [ ] Feedback-Mechanismus aktiv (optional für Launch)
- [ ] Prompt-Versioning implementiert

### Documentation

- [ ] Alle Compliance-Docs in /docs/compliance/
- [ ] Implementation Log aktuell
- [ ] Decisions Made dokumentiert
- [ ] Open Issues leer oder priorisiert

## Laufender Betrieb Checkliste

### Monatlich

- [ ] Review aller AI-generierten Antworten (5% Stichprobe)
- [ ] Prüfung der Logs auf Daten-Leaks
- [ ] Verifizierung aller DPAs (aktuell)
- [ ] Überprüfung der Aufbewahrungsfristen

### Quartalsweise

- [ ] Vollständiger Code-Audit gegen KI-Protokoll
- [ ] Prompt-Überprüfung und Optimierung
- [ ] Review der Datenschutzerklärung (Aktualisierung falls nötig)
- [ ] VVT aktualisieren (bei Änderungen)
- [ ] Security-Review

### Jährlich

- [ ] Vollständiger Security-Audit
- [ ] Test des Incident Response Plans
- [ ] Review und Update des gesamten KI-Protokolls
- [ ] Überprüfung aller rechtlichen Dokumente

---

## Bei Anfragen Checkliste

### Betroffenenrechte (Art. 15-22 DSGVO)

- [ ] Anfrage dokumentieren
- [ ] Identität des Anfragenden verifizieren
- [ ] Innerhalb 30 Tage antworten
- [ ] Daten bereitstellen (Export) oder löschen
- [ ] Vorgang dokumentieren

### Datenschutzverletzungen (Art. 33/34 DSGVO)

- [ ] Vorfall dokumentieren
- [ ] Umfang ermitteln
- [ ] Risiko bewerten
- [ ] Aufsichtsbehörde kontaktieren (innerhalb 72h)
- [ ] Betroffene benachrichtigen (falls hohes Risiko)
- [ ] Maßnahmen zur Behebung umsetzen

---

## Testing-Checkliste

### Compliance-Tests

- [ ] AI-Disclaimer ist auf allen Antworten sichtbar
- [ ] Datenschutzerklärung ist erreichbar
- [ ] User kann Daten exportieren
- [ ] User kann Account löschen
- [ ] Cookie-Banner wird angezeigt
- [ ] AI-Prompt enthält Compliance-Regeln
- [ ] Keine sensiblen Daten in Error-Logs
- [ ] Footer-Links funktionieren

### Edge-Case-Tests für AI-Responses

- [ ] Positive Reviews
- [ ] Negative Reviews
- [ ] Neutrale Reviews
- [ ] Reviews mit Gesundheitsdaten (Warnung?)
- [ ] Reviews mit Rechtsfragen (Warnung?)
- [ ] Beleidigende Reviews
- [ ] Reviews ohne Text (nur Sterne)
